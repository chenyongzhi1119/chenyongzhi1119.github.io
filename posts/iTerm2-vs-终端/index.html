为什么要用 iTerm2 而不是 macOS 自带终端

发表于2026-03-21|更新于2026-03-21|工具
|浏览量:

macOS 自带的 Terminal（终端）能完成大多数命令行任务，但如果你每天都在用命令行，iTerm2 在体验、功能和定制化上会让你感受到明显的差距。

1. 分屏与标签管理

macOS 终端支持多标签，但分屏功能极其有限。

iTerm2 支持：

  • 水平 / 垂直任意分屏（⌘D / ⌘⇧D）
  • 标签内嵌套多个面板
  • 拖拽调整面板大小

左边跑服务器日志，右边编辑配置文件，下方执行命令——全在一个窗口完成，不需要反复切换应用。

2. 强大的搜索与高亮

功能 终端 iTerm2
全文搜索 ❌ ✅ 实时高亮
正则搜索 ❌ ✅
触发器高亮 ❌ ✅ 自定义规则

⌘F 打开搜索栏，支持正则表达式，匹配内容即时高亮。调试日志时非常实用。

3. 自动补全与命令历史

  • ⌘;：打开自动补全弹窗，基于历史命令提示
  • ⌘⇧H：粘贴历史记录，可搜索所有历史复制内容
  • ⌘⌥B：Instant Replay——像录像回放一样回溯终端输出，即使内容已滚出屏幕也能找回

4. 配置文件（Profiles）

iTerm2 支持多个配置文件（Profiles），每个 Profile 可以独立设置：

  • 字体、字号、配色方案
  • 默认工作目录
  • 环境变量
  • 启动命令（如自动 SSH 到某台服务器）

为每个项目或服务器创建一个 Profile，一键打开就自动连接到对应环境，省去重复输入的麻烦。

5. 触发器（Triggers）

根据终端输出内容自动执行动作，这个功能系统终端完全没有：

  • 输出包含 ERROR → 自动高亮 + 发送系统通知
  • 匹配特定 URL → 自动打开浏览器
  • 匹配 IP 地址 → 高亮显示

在 Preferences → Profiles → Advanced → Triggers 中配置。

6. Shell 集成

安装 Shell Integration 后（iTerm2 → Install Shell Integration）：

  • 每条命令的执行时间自动标注在右侧
  • 上一条命令的退出状态用颜色标识（绿色成功 / 红色失败）
  • ⌘⇧Up/Down 快速跳转到上/下一条命令的起始位置

这几个功能合在一起，让你在翻看长输出时效率大幅提升。

7. 其他实用功能

功能 说明
鼠标选中自动复制 选中即复制，无需 ⌘C
内置密码管理器 存储常用密码，一键粘贴
广播输入 同时向多个面板发送相同命令
静默通知 长任务完成后系统通知提醒
配色主题 兼容 Solarized、Dracula、Nord 等主流主题
Visor 模式 最小化到状态栏，热键随时呼出

总结

如果你每天都在使用命令行，iTerm2 相比系统终端能显著提升效率。分屏、搜索、Profiles、Shell Integration 这四个功能就已经值得迁移。

下载地址：https://iterm2.com（免费开源）

推荐搭配 Oh My Zsh + Starship 提示符，打造完整的命令行工作流。
