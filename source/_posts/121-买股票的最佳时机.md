## 题目

[题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/)

![alt text](/images/121.png)

## 思路

1. 买股票的问题一般考虑使用动态规划
2. 需要先确定 dp 数组的定义，dp[i][0 or 1]表示第（i+1）天时，处于持有股票（1）或者没有持有股票（0）时的金额
3. base case 情况，dp[0][0] = 0, dp[0][1] = - prices[0]
4. 递推公式：
   - dp[i][0] = max(dp[i-1][0], dp[i-1][1] + prices[i])
   - dp[i][1] = max(-prices[i], dp[i-1][1])
5. 最终的结果是 dp[n-1][0]

## 代码实现

```go
func maxProfit(prices []int) int {
    // n 表示有多少天
    n := len(prices)
    // dp 是一个 n*2 的数组
    // dp[i][0 or 1] 表示 第（i+1）天时，持有股票（1）or 没有持有股票（0）时的金额
    dp := make([][2]int, n)

    for i := 0; i<n; i++ {
        // base case
        if i == 0 {
            dp[i][0] = 0
            dp[i][1] = -prices[i]
            continue
        }
        // 递推公式
        dp[i][0] = max(dp[i-1][0], dp[i-1][1] + prices[i])
        dp[i][1] = max(- prices[i], dp[i-1][1])
    }
    // 最终结果是在第 n 天时，没有持有股票的金额
    return dp[n-1][0]
}

```
