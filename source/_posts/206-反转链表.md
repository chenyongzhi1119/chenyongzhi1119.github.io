## 思路

考虑使用递归法解决

1. 明确定义的递归函数的输入和输出
2. 寻找递归函数终止条件
3. 对一每一次递归的处理逻辑

## 代码实现

``` go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */

 // 使用递归法
 // 需要明确递归函数的定义和返回值
 // reverseList(head *ListNode) *ListNode 表示输入一个链表
 // 头结点 head，放回反转后的新头结点 last
func reverseList(head *ListNode) *ListNode {
    // head == nil 是针对空链表的情况
    // head.Next == nil 是对于一般链表的情况，找到链表的最后一个节点
    if head == nil || head.Next == nil {
        return head
    }

    // 通过递归实现
    // last 是最后一个节点
    // 此时 head 是 last 的前一个节点
    last := reverseList(head.Next)

    // head.Next 表示 head 的下一个节点
    // head.Nexy.Next 表示 head 的下一个节点的指向
    head.Next.Next = head

    // 将 head 的指向变为 nil
    head.Next = nil

    // 传递反转后的新的头结点最初调用的函数
    return last

}
```
