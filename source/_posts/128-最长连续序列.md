## 题目

[LeetCode 128. 最长连续序列](https://leetcode.cn/problems/longest-consecutive-sequence/description/?envType=study-plan-v2&envId=top-100-liked)

![alt text](/images/128.png)

## 思路

1. 要找连续子序列，首先第一步是找到连续子序列的头部元素。
2. 为了能够在$O(n)$复杂度下，快速判断一个元素是不是头部元素，考虑使用 set 数据结构
3. 找到头部元素后，逐次加一，判断新的元素是不是在 set 中，如果在，则更新长度。如果不在，直接跳过。

## 代码实现

```go
func longestConsecutive(nums []int) int {
    set := make(map[int]struct{}) // map 的 key 是 int，vlaue 是 struct{}类型
    for _, num := range nums {
        set[num] = struct{} {} // struct{}是数据类型，后面的{}表示一个实例
    }

    res := 0

    for num := range set {
        if _, found := set[num-1]; found {
            // num 不是连续子序列的头一个元素，跳过
            continue
        }
        // num 是连续子系列的首个元素
        curNum := num
        curLen := 1
        for {
            if _, found := set[curNum+1]; !found {
                break
            }
            curNum++
            curLen++
        }
        res = max(res, curLen)

    }
    return res
}
```
