160. 相交链表

Posted on 2025-07-23 Edited on 2026-03-20 In LeetCode

题目题目链接

思路

难点: 每个链表的长度不一定相同，如何找到相交点？
思路: 使用双指针法，两个指针分别遍历两个链表，当一个指针到达链表末尾时，跳转到另一个链表的头部。这样可以确保两个指针在相同的步数后相遇。
复杂度分析:
时间复杂度: O(m + n)，其中 m 和 n 分别是两个链表的长度。
空间复杂度: O(1)，只使用了常数级的额外空间。

注意事项:
注意处理链表长度不相等的情况。

边界情况: 如果两个链表没有相交点，返回 null。

121. 买股票的最佳时机

Posted on 2025-07-16 Edited on 2026-03-20 In LeetCode

题目题目链接

思路

买股票的问题一般考虑使用动态规划
需要先确定 dp 数组的定义，dp[i][0 or 1]表示第（i+1）天时，处于持有股票（1）或者没有持有股票（0）时的金额
base case 情况，dp[0][0] = 0, dp[0][1] = - prices[0]
递推公式：
dp[i][0] = max(dp[i-1][0], dp[i-1][1] + prices[i])
dp[i][1] = max(-prices[i], dp[i-1][1])

最终的结果是 dp[n-1][0]

128. 最长连续序列

Posted on 2025-07-15 Edited on 2026-03-20 In LeetCode

题目LeetCode 128. 最长连续序列

思路

要找连续子序列，首先第一步是找到连续子序列的头部元素。
为了能够在$O(n)$复杂度下，快速判断一个元素是不是头部元素，考虑使用 set 数据结构
找到头部元素后，逐次加一，判断新的元素是不是在 set 中，如果在，则更新长度。如果不在，直接跳过。
