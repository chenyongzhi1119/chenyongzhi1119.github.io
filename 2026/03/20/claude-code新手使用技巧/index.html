Claude Code 新手使用技巧

Posted on 2026-03-20 In 工具

Claude Code 是 Anthropic 推出的 AI 编程助手 CLI 工具，能够直接在终端中读写代码、执行命令、操作 Git 等。本文整理了新手上手时最实用的技巧。

技巧二：善用 CLAUDE.md 配置项目上下文

在项目根目录创建 CLAUDE.md，Claude Code 每次启动都会自动读取它。可以写入：

  • 项目架构说明
  • 常用命令（构建、测试、部署）
  • 代码规范和注意事项

这样就不用每次重复解释项目背景了。

技巧三：用 / 斜杠命令提高效率

Claude Code 内置了一批斜杠命令，常用的有：

命令 作用
/clear 清空当前对话上下文
/compact 压缩历史消息，节省上下文空间
/init 分析项目并自动生成 CLAUDE.md
/model 切换使用的模型
/help 查看所有可用命令

上下文接近限制时，先用 /compact 压缩，再继续工作。

技巧四：逐步确认，避免误操作

Claude Code 会主动执行命令（包括删除文件、推送代码等）。新手建议：

  1. 每次操作前看清楚它要做什么，工具调用会显示具体命令
  2. 涉及 Git push、删除文件等高风险操作时，它会默认先询问确认
  3. 可以随时按 Esc 中断正在执行的操作

小结

场景 技巧
不知道命令 直接描述意图
重复解释项目背景 写好 CLAUDE.md
上下文满了 用 /compact 压缩
怕误操作 拆小步骤 + 确认后再执行

Claude Code 本质上是一个懂代码的协作者，越清晰地表达需求，它给出的结果越准确。多用、多实验，很快就能找到适合自己的工作方式。
